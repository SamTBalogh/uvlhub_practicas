# uvlhub.io

Repository of feature models in UVL format integrated with Zenodo and flamapy following Open Science principles. Developed by DiversoLab.

This fork is used as a personal **testing sandbox** by **Samuel Tamayo Balogh**, student at the **University of Seville**, to validate development, workflows, automation, deployment and documentation conventions without impacting upstream development.

## Official documentation
You can consult the official documentation of the project at [docs.uvlhub.io](https://docs.uvlhub.io/).

## Development workflow
This repository follows a **no Pull Requests** workflow (local merge + push) with short-lived branches and Conventional Commits. For the operational contribution flow and rules, see:

- `CONTRIBUTING.md`
- `docs/BRANCHING_STRATEGY.md`
- `docs/COMMIT_CONVENTION.md`
- `docs/CI_CD_WORKFLOWS.md`
- `docs/ISSUES_TEMPLATES.md`
- `docs/LABELS_GUIDE.md`
